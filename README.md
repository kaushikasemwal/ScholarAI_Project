# ScholarAI — AI-Powered Study Assistant

Backend API for ScholarAI. Built with FastAPI + PyTorch + HuggingFace Transformers.

**Frontend:** Deployed on GitHub Pages  
**Course:** Advanced Topics in Machine Learning

## ML Pipeline
- Sentence-BERT → Autoencoder → BART summarization
- NLP-based quiz generation (spaCy + TF-IDF)
- gTTS audio narration
- MoviePy slide-style explainer video
