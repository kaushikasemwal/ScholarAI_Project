Autoencoder Architecture — Advanced ML Component

Encoder
Input: 768-dim (SBERT)
Dense 512 + ReLU
Dense 256 + ReLU
Latent: 128-dim ✦
⟷
Decoder
Latent: 128-dim ✦
Dense 256 + ReLU
Dense 512 + ReLU
Output: 768-dim

The autoencoder compresses 768-dimensional Sentence-BERT embeddings to a 128-dimensional latent space, removing redundancy while preserving semantic meaning. This compressed representation is then fed into the BART summarizer, improving coherence and reducing noise in the generated summaries.
