Autoencoder Architecture — Advanced ML Component

Encoder
Input: 384-dim (BGE)
Dense 512 + ReLU
Dense 256 + ReLU
Latent: 128-dim ✦
⟷
Decoder
Latent: 128-dim ✦
Dense 256 + ReLU
Dense 512 + ReLU
Output: 384-dim

The autoencoder compresses BGE embeddings to a 128-dimensional latent space, removing redundancy while preserving semantic meaning. This compressed representation feeds into Pegasus for cleaner, more coherent summaries.
